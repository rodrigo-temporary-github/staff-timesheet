# staff-timesheet

### How to build the app:

`./gradlew build`

### How to run the app:

`java -cp build/libs/staff-timesheet-0.1.0.jar com.stafftimesheet.Main`

### How to run unit tests only:

`./gradlew test`

### Test report is generated at: 

`build/reports/tests/test/index.html`
